# Rev C Prototype Test Plan

Use the Rev B bring-up plan after Rev C becomes routed and DRC-clean.

Current Rev C is not ready for electrical bring-up because it is not routed.

Before any prototype order:

- DRC must be clean or every exclusion must be justified.
- Gerbers must be visually inspected.
- Schematic-to-PCB parity must be checked.
- Lock power path must be reviewed.
- Buck layout must be reviewed.
- RS-485 A/B routing and protection placement must be reviewed.
