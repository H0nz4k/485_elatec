# KiCad: TWN4 access system

This is a concept KiCad project for two related hardware blocks:

- reader-side TWN4 carrier / door module
- Raspberry Pi-side RS-485 gateway / HAT

The schematic is intentionally Rev A concept level. Footprints, exact regulator choice, lock current, BLE antenna keepout, COM1 polarity, and OSDP firmware timing must be verified before PCB layout.
