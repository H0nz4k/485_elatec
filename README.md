# 485_elatec

Hardware concept for an ELATEC TWN4 MultiTech 3 BLE carrier board with RS-485 / OSDP, wide-range DC input, door lock outputs, and a Raspberry Pi-side RS-485 gateway.

## Current contents

- `analyza_twn4_carrier.md` - design analysis and architecture notes
- `schema_twn4_carrier_rev_a.md` - first schematic-level design proposal
- `rpi_gateway_hw.md` - Raspberry Pi RS-485 gateway / HAT notes
- `kicad/twn4_access_system/` - KiCad concept schematic for the reader carrier and RPi gateway

The design is currently Rev A concept-level work. It still needs component selection, footprint assignment, detailed ERC review, PCB layout, and bench validation before hardware production.
